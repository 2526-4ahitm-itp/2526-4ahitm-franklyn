Franklyn Sentinel - Portable

Requirements:
- Linux with glibc 2.34 or newer (e.g. Ubuntu 22.04+, Debian 12+,
  Fedora 35+, RHEL/Rocky/Alma 9+). X11 and Wayland are both supported;
  Wayland screen capture additionally needs PipeWire installed on the host.

How to run:
- run ./bin/franklyn
