## Highlights

- WebSocket streaming end-to-end between sentinel, server, and proctor.
- Proctor image streamer with sentinel selection and base64 frames.
- Release automation matured with docker, k8s, and CI consolidation.

## Added

- Sentinel WebSocket client plus screen capture and fragmentation for large frames.
- Server WebSocket communication, cache/frame listener work, and protocol docs.
- Proctor UI for sentinel selection and streamed image display.
- Docs: Nix/CI guidance, WebSocket lifecycle, media spec, and sprint reviews.

## Changed

- Server stack refactor from Kotlin to Java with coverage in CI.
- CI: merged PR checks, archive workflow fixes, and docker image conventions.
- Deployment: nix-generated k8s manifests and docker build/test scripts.

## Fixed

- WebSocket frame RSV handling and PNG encoding dimensions.
- Server startup schema creation for fresh databases.
