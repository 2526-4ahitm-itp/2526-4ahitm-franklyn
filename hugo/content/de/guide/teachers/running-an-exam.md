## Test starten

1. Melde dich bei Franklyn Proctor unter `franklyn.htl-leonding.ac.at/proctor` an.
2. Erstelle einen neuen Test oder wähle einen bestehenden aus und starte ihn.
   ![Proctor Dashboard](/Create-Exam.png)
3. Ein PIN wird angezeigt. Teile ihn mit deinen Schülern, damit sie beitreten können.

## Überwachung

Sobald Schüler beigetreten sind, kannst du sie im Dashboard überwachen:

- **Übersicht**: ein Raster aller aktiven Schülerbildschirme auf einen Blick
  ![Proctor Monitoring](/Proctoring.png)
- **Detailansicht**: klicke auf einen Schüler, um seinen Bildschirm vergrößert anzuzeigen
