## Code

<p class="text-line">Code highlighting works with inline <code>code</code> and code blocks:</p>

<pre><code><span class="code-keyword">func</span> <span class="code-function">main</span>() {
    <span class="code-variable">fmt</span>.<span class="code-function">Println</span>(<span class="code-string">"Hello, World!"</span>)
}</code></pre>

<p class="text-line">Use the <span class="highlight">highlight</span> class for emphasis.</p>
