## Features

### Sentinel GStreamer Rewrite & Wayland Support

Sentinel was completely rewritten to use GStreamer. This provides a more robust recording API and introduces native support for Wayland displays.

### Exams Dashboard & PIN Login

An exams dashboard was added to the Proctor UI, allowing teachers to easily manage and start proctoring sessions for tests. In addition, exams now generate a unique PIN. Sentinels are required to join exams using this PIN via the `--pin` flag.

### Sentinel Packaging Improvements

Sentinel is now distributed as a tarball containing the executable binary, application icons, and a desktop entry instead of just a raw binary. This improves the overall installation and desktop integration experience for students.
